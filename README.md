# hlhouseerp2csv

這是一個 Chrome 擴充功能，用來攔截 hlhouseerp 查詢時送出的條件，並在頁面上提供一個浮動下載面板，將查詢結果批次抓回並匯出成 CSV。

## 功能說明

- 自動攔截 hlhouseerp 網站上的查詢條件
- 在頁面右側顯示浮動下載按鈕
- 支援批次下載多頁資料
- 下載時顯示進度圈與狀態文字
- 匯出 UTF-8 BOM 的 CSV，避免 Excel 開啟中文亂碼
- 依不同頁面自動套用對應欄位格式

## 專案結構

- `manifest.json`
  Chrome Extension Manifest V3 設定檔，定義權限、背景腳本與 content script。
- `background.js`
  監聽網站的查詢請求，攔截 POST payload，並傳給頁面中的 content script。
- `content.js`
  注入頁面 UI，接收查詢條件、執行批次下載、顯示進度，最後輸出 CSV。

## 使用方式

### 1. 載入到 Chrome

1. 打開 Chrome。
2. 進入 `chrome://extensions/`。
3. 右上角打開「開發人員模式」。
4. 點擊「載入未封裝項目」。
5. 選擇這個專案資料夾：
   `/Users/c2leb/Codes/hlhouseerp2csv`
6. 載入完成後，Chrome 就會啟用這個擴充功能。

### 2. 在網站中使用

1. 打開 hlhouseerp 網站的查詢頁面。
2. 先在網站內設定查詢條件。
3. 點擊網站原本的「搜尋」，讓系統送出查詢。
4. 擴充功能會攔截這次查詢條件，並在右側顯示浮動下載按鈕與面板。
5. 點擊面板中的「開始下載」。
6. 擴充功能會自動分頁抓取資料並下載成 CSV。

## 操作畫面說明

- 右側綠色按鈕：開啟或收合下載面板
- 等待條件中：代表尚未攔截到查詢條件，請先在網站按一次搜尋
- 開始下載：已攔截到條件，可以開始匯出
- 資料抓取中：正在逐頁抓資料
- 圈圈進度：顯示目前抓取進度
- 完成：表示資料已整理並開始下載 CSV

## 重新載入擴充功能

如果你修改了程式碼，需要重新載入擴充功能：

1. 回到 `chrome://extensions/`
2. 找到「房地產終極下載器」
3. 點擊重新整理按鈕
4. 回到網站頁面重新整理一次

## 權限說明

這個專案目前使用以下權限：

- `webRequest`
  用來攔截 hlhouseerp 的查詢請求
- `host_permissions`
  允許擴充功能在 `hlhouseerp.com.tw` 網域下運作

## 注意事項

- 必須先在網站內按過一次搜尋，擴充功能才會知道目前查詢條件。
- 如果下載結果不對，通常是因為查詢條件已改變但尚未重新搜尋。
- 每次改完程式碼後，都要重新載入 Chrome 擴充功能。
- 若下載過程中發生錯誤，可打開瀏覽器開發者工具查看 Console 訊息。

## 適合的使用情境

- 匯出成交行情資料
- 匯出委託物件資料
- 將網站查詢結果整理成 CSV 再進一步分析

## 開發備註

這是一個不需要編譯流程的純 JavaScript Chrome Extension 專案。

開發時只要直接修改以下檔案即可：

- `manifest.json`
- `background.js`
- `content.js`

修改後重新載入 Chrome 擴充功能，就能看到最新版本。
